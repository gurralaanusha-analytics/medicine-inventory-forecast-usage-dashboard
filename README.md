# Medicine Inventory Forecast & Usage Dashboard

## Overview
This project was developed as part of my Master’s in Analytics. It focuses on analyzing medicine inventory data and forecasting future demand to support better inventory management and reduce shortages.

## Objective
The goal of this project is to build a system that helps predict medicine usage trends and improve decision-making in inventory planning.

## Project Type
Group Project (Master’s Research Project)

## Team Contribution
This project was completed as part of a team during my Master’s program.

My contributions included:
- Data cleaning and preprocessing
- Exploratory data analysis
- Forecasting model implementation
- Supporting dashboard insights

## Tools & Technologies
Python, Data Analysis, Forecasting Techniques, Dashboard Development
